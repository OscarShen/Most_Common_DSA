# Sort

排序算法可以说是一种非常非常常用的算法！我们这里都是按照升序去排列。

我们这一库中实现的的排序算法有：

1. 冒泡排序
2. 选择排序
3. 插入排序
4. 归并排序
5. 快速排序
6. 就地堆排序

## 理解排序

冒泡排序的核心思想是：在一趟遍历的过程中，一定可以把数组中最大的元素交换到数组的最末端，那么只需要反复遍历冒泡，数组终将有序！

选择排序与冒泡排序在思想上是差不多的，也是每次都将最大元素选出然后放置在数组最末端，反复执行后数组终将有序。

插入排序则是每次取出数组中无序的部分的第一个元素，插入到有序部分的合适位置，这样反复后数组终将有序。

上面三种排序算法的时间复杂度在最坏情况下都将达到O(n^2);而下面的三种排序算法则要快的多~

归并排序和快速排序可以对比着来理解，归并排序的主要时间消耗在合并操作，而快速排序的主要时间消耗在划分操作；

堆排序和选择排序很像！它是非常巧妙地利用了优先级队列这种数据结构来减少了选择排序中选择操作所需要花费的时间，从而大大减少了排序需要的时间；
